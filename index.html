Sobre

Este trabalho tem como objetivo demonstrar de forma teórica e prática como as diretrizes de acessibilidade podem ser inseridas em aplicações web de forma a garantir que o usuário portador de deficiência visual possa utilizá-la sem eventuais problemas. O principal objetivo deste trabalho é desenvolver uma aplicação web que ofereça modelos com diretrizes de acessibilidade para desenvolvedores. Como objetivos específico, podemos ressaltar: Fornecer informações para desenvolvedores web sobre como deixar um site acessível para pessoas com deficiências visuais. Permitir através das informações fornecidas que se diminua os índices de exclusão de deficientes visuais na web.
